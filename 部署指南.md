# 🚀 部署指南 — 3 种方式任选

## ⚡ 方式一：Netlify Drop（最快，30秒，推荐）

1. 打开 👉 https://app.netlify.com/drop
2. 打开这个文件夹：`C:\Users\Administrator\Desktop\booking-web`
3. **把整个文件夹拖进网页**
4. 等待 10 秒，得到一个网址，格式类似：`https://xxx.netlify.app`
5. 把这个网址发给朋友，他们打开就能预约！

✅ 完全免费，不需要注册（如果想管理再去注册）
✅ 朋友随时能访问，不需要同一 WiFi

---

## 🐙 方式二：GitHub Pages（永久免费，最稳定）

### 第一步：在 GitHub 新建仓库
1. 打开 👉 https://github.com/new
2. 仓库名填：`booking`（随便取）
3. 选 **Public**
4. 点 **Create repository**

### 第二步：上传文件
1. 在仓库页面点 **uploading an existing file**
2. 把 `C:\Users\Administrator\Desktop\booking-web\` 里的**所有文件**拖进去（不包括文件夹本身）
3. 点 **Commit changes**

### 第三步：开启 Pages
1. 仓库页面点 **Settings**
2. 左侧点 **Pages**
3. **Source** 选 **Deploy from a branch**
4. **Branch** 选 **main** 和 **/ (root)**
5. 点 **Save**
6. 等待 1 分钟，得到网址：`https://你的用户名.github.io/booking/`
7. 发给朋友即可！

---

## 📱 方式三：本地服务器（仅同一 WiFi）

1. 双击运行 `start-server.bat`（已自动创建）
2. 把你的内网 IP 发给朋友：`http://你的IP:8080`
3. ⚠️ 你电脑关机后朋友无法访问

---

## 📱 如何在手机上管理？

### 方式 A：用手机浏览器打开 admin.html

把网址发给朋友后，你自己也打开：
```
https://你的网址/admin.html
```

用手机浏览器（Safari/Chrome）打开，就能在手机上管理：
- ✅ 添加/编辑服务
- ✅ 上传场地图片
- ✅ 查看预约记录
- ✅ 修改预约状态（待付款→已付款）
- ✅ 设置折扣规则

### 方式 B：手机添加到桌面（像 App 一样用）

1. iPhone（Safari）：打开 `admin.html` → 点底部分享按钮 → **添加到主屏幕**
2. 安卓（Chrome）：打开 `admin.html` → 点右上角 ⋮ → **安装应用**

## ❓ 遇到问题？

**Q：Netlify 拖不进去？**
A：换 Chrome 浏览器，或者直接用方式二（GitHub Pages）

**Q：GitHub Pages 打不开？**
A：等 2 分钟再试，第一次部署需要一点时间

**Q：手机管理数据存在哪？**
A：存在浏览器 localStorage，换手机/换浏览器数据不共享。如需多设备同步，需要后端数据库（可以后续升级）
